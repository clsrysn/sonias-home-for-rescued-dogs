# How to Update Dog Information - Simple Guide for Ms. Sonia

## 🚨 **IMPORTANT UPDATE** - New System

**We've moved from files to a web-based database!** You no longer edit files directly.

## How to Add/Update a Dog

### Adding a New Dog (Web Dashboard)
1. Go to your admin login URL (contact development team for access)
2. Click **"Add New Dog"** button
3. Fill in the dog information form
4. Click **"Save Dog"**

### Updating an Existing Dog (Web Dashboard)
1. Go to your admin dashboard
2. Find the dog in the list
3. Click the **Edit button** (pencil icon)
4. Change any information you want
5. Click **"Update Dog"**

## Image Management

### Adding Dog Photos
1. Upload images to the **public/dogs/** folder
2. Use the web dashboard to set the image path:
   - Main image: `/dogs/dog-name.jpg`
   - Gallery: `/dogs/dog-name/` (folder name)

### Supported Image Formats
- **JPG, PNG, GIF, WebP**
- **Recommended size:** 800x800px for main photos
- **File naming:** Use lowercase, no spaces

### Adoption Status
- **Available** = Shows "Support Care" button
- **Adopted** = Shows "Adopted" badge, hides adoption button
